Projekte

Unity ML-Agents

Programmiersprache: C#

Technologien: Unity, Blender, Reinforcement Learning

Eine Studienarbeit, mit dem Ziel das Lernverhalten von Neuralen Netzen (Reinforcement Learning) mit unterschiedlicher Komplexität bei gleichbleibender Aufgabenstellung zu untersuchen. Hierfür wurde in der Spiel-Engine Unity ein simples Auto und eine Rennstrecke entworfen. Anhand von Unity ML-Agents das maschinelle Lernen ermöglicht. Weitere benötigte Assets wurden mit Hilfe von Blender modelliert.

DIY Button Box

Programmiersprache: C, C++

Technologien: Microcontroller, Fusion360, 3D-Druck, Schaltkreise

Eine kostengünstige DIY Button Box mit selbst entworfenem Schaltkreis. Es handelt sich hierbei um eine Anordnung von Tastern, Potentiometern und Drehgebern (eng. "rotary encoder"). Diese werden von einem Arduino Microcontroller gesteuert um Signale an einen verbundenen Rechner zu senden. Der Rechner erkennt diesen wiederum als Eingabegerät und kann als Controller bei Rennsimulationen genutzt werden. Das Gehäuse wurde in Fusion360 entworfen und mit einem 3D-Drucker ausgedruckt.

Twitter Bot

Programmiersprache: Python

Technologien: Raspberry Pi, Linux, Cron

Ein Twitter Bot welcher periodisch einen in ASCII Zeichen generierten Ladebalken anzeigt. Dieser Ladebalken repräsentiert den Fortschritt zwischen einem Start- und Ziel-Datum. In diesem Fall wurde dies genutzt um den Release der zweiten Staffel der Serie "Arcane" darzustellen. In einem Python Script wurden die Berechnungen durchgeführt und der ASCII Ladebalken berechnet. Auch wurde eine Logging Funktion eingebaut, um bei Fehlern die Ursache ausfindig zu machen. Der Bot wurde mit der Twitter-API bedient und auf einem Raspberry Pi gehosted. Zur Automatisierung wurde Cron verwendet.

TNT-Lad

Programmiersprache: C#, PHP

Technologien: Unity, Blender, MySQL, Datenbanken

Ein 3D-Remake des Klassikers "Bomberman" in Unity. Dieser wurde in einem Team von 4 Studenten entwickelt. Neben den bekannten Features des Originals hat dieses Remake noch einen eigenen Level-Editor. Erstellte Level können, dank einer angebundenen Datenbank, in eine Cloud hochgeladen werden um sie mit allen Spielern zu teilen.
